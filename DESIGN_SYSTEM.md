# Design Style

**Contemporary Real Estate Editorial Interface**

A hybrid of:

* Swiss International Style grid discipline
* Airbnb product clarity
* Apple-like spatial reduction
* Contemporary PropTech dashboard design
* Architectural visualization aesthetics
* Luxury real-estate editorial publishing
* Soft Scandinavian modernism
* Webflow SaaS conversion architecture

This is not a real-estate website.

This is a **property marketplace presented as an architectural magazine.**

---

# Concept

### Core Narrative

**"Property discovery should feel premium, effortless, and trustworthy."**

The design removes traditional marketplace clutter and replaces it with:

* calm
* space
* hierarchy
* confidence

The user should feel:

* financially secure
* visually comfortable
* guided rather than sold

# Rules

### Mandatory

* Use oversized headlines.
* Use architectural photography only.
* Maintain generous whitespace.
* Prioritize imagery over interface.
* Use Neo-Grotesk typography.
* Keep UI chrome minimal.
* Preserve editorial pacing.
* Use rounded corners consistently.
* Maintain low visual noise.
* Create calm visual flow.

---

# Anti-Rules

### Forbidden

* Glassmorphism
* Neumorphism
* Corporate SaaS gradients
* Startup illustrations
* Cartoon graphics
* Excessive iconography
* Bright saturated palettes
* Dark cyber aesthetics
* Brutalist typography
* Generic stock photography
* Pinterest luxury clichés
* AI-generated decorative abstractions
* Dense dashboard interfaces
* Overuse of shadows

---

# Quality Standards

### Historical Authenticity

Swiss editorial discipline blended with contemporary PropTech restraint.

### Typography Coherence

Single Neo-Grotesk family with disciplined scale hierarchy.

### Color Coherence

Warm neutrals dominate.
Orange appears only as conversion catalyst.

### Narrative Coherence

Every section must support:
Trust → Discovery → Validation → Conversion.

### Editorial Coherence

Should resemble a luxury property publication before it resembles software.

### Brand Coherence

Premium.
Calm.
Architectural.
Trustworthy.

### Final Litmus Test

If a section feels like:

* a startup template,
* a Dribbble concept,
* a generic Webflow clone,
* an AI-generated landing page,

it fails the system and must be redesigned.
